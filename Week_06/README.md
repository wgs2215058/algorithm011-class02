本周是动态规划： 
但是有几个要点还是很重要：
动态规划和递归或者分治没有根本上的区别（关键看有无最优的子结构），共性：找到重复子问题，差异性：最优子结构、中途可以淘汰次优解

关键点：

最优子结构 opt[n] = best_of(opt[n-1], opt[n-2], …)
储存中间状态：opt[i]
递推公式（状态转移方程或者DP方程）
